# 🛠️ Skills do Claude Code — cole aqui

Esta é a pasta onde o Claude Code descobre **skills** automaticamente.

## Como colar uma skill

Cada skill é uma **pasta** com (no mínimo) um arquivo `SKILL.md` dentro:

```
.claude/skills/
├── nome-da-skill-1/
│   └── SKILL.md
├── nome-da-skill-2/
│   ├── SKILL.md
│   └── (outros arquivos de apoio)
```

➡️ Cole a **pasta inteira** de cada skill aqui dentro. Depois me avise — eu confirmo que foram
reconhecidas e passamos a usá-las nas fases de UI/UX.

## "Isso é uma skill ou só referência?"

- Tem um arquivo **`SKILL.md`** dentro? → é skill do Claude Code → **cole aqui**.
- É um PDF / print / Figma / doc solto? → é material de referência → cole em
  `docs/checkout/referencias/ux/` (UI/UX) ou `docs/checkout/referencias/api-pix/` (API PIX).
